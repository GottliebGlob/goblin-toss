## Goblin Toss 

![Example 1](https://github.com/GottliebGlob/goblin-toss/blob/main/example.png?raw=true "Example 1")

Goblin Toss is a simple coin toss gambling game build with Next.js and Solidity. 

## Getting started

```bash

yarn dev
```
